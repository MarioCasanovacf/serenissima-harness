## Objetivo

Probar que un agente de OTRO motor (tú) puede seguir el protocolo del harness de punta a
punta: leer el blackboard, reclamar una tarea respetando la compuerta de cascada, tomar y
liberar locks, producir un artefacto, dejar evidencia ReContext y hacer hand-off al
verificador. Con esto validamos la **portabilidad del NLAH** y rompemos el monocultivo
cognitivo (que ningún pilar del sistema dependa de un solo tipo de modelo).

## Pasos exactos (ejecútalos en la terminal del workspace, en este orden)

```bash
# 0. Sitúate en la raíz del workspace "Universal Harness" e identifícate
export CLAUDE_HARNESS_AGENT_ID=gemini-runner

# 1. Lee el tablero (¿qué hay claimable? ¿qué está gated por cascada?)
python3 .harness/bin/blackboard.py status

# 2. Reclama TU tarea (será rechazada solo si sus dependencias no están done — eso sería un hallazgo)
python3 .harness/bin/blackboard.py claim T-002 --agent gemini-runner

# 3. Toma el lock del archivo que vas a escribir
python3 .harness/bin/lock.py acquire ".harness/logs/gemini_contract_report.md" --holder gemini-runner --task T-002

# 4. Anuncia que trabajas
python3 .harness/bin/blackboard.py update T-002 --status in_progress --note "gemini-runner: ejecutando checklist de contrato" --agent gemini-runner
```

5. **Escribe el reporte** `.harness/logs/gemini_contract_report.md` con:
   - **Checklist de protocolo**: cada paso de `gemini.md` §2B y §4C (leer blackboard,
     locks, ciclo, liberar y actualizar) marcado OK/FALLO, con el output real del comando.
   - **3 fricciones** del harness vistas desde la perspectiva Gemini (¿qué instrucción fue
     ambigua? ¿qué comando fue incómodo? ¿qué faltó?) + una mejora propuesta por fricción.
     Esto alimenta la auditoría evolutiva T-006.
   - **ReContext (gemini.md §4B)**: copia en `.harness/recontext_evidence.md` los bloques
     EXACTOS de `gemini.md` §4B y del `protocol` de `blackboard.json` que usaste, con el
     formato indicado en ese archivo.

```bash
# 6. Registra el artefacto y entrega al verificador (NUNCA te marques done a ti mismo)
python3 .harness/bin/blackboard.py update T-002 --artifact ".harness/logs/gemini_contract_report.md" --agent gemini-runner
python3 .harness/bin/blackboard.py handoff T-002 --to-role verifier --note "reporte y evidencia ReContext listos; replay: cat .harness/logs/gemini_contract_report.md" --agent gemini-runner

# 7. Libera tu lock (cero residuos)
python3 .harness/bin/lock.py release ".harness/logs/gemini_contract_report.md" --holder gemini-runner

# 8. Estado final para el entregable
python3 .harness/bin/blackboard.py status
```
