# Gemini for Science (Google I/O 2026)

This document summarizes the announcement and key details of **Gemini for Science**, introduced by Google at Google I/O 2026.

---

## 1. Initiative Overview
**Gemini for Science** is a collection of AI-powered tools and agentic skills hosted in Google Labs, designed to automate and accelerate stages of the scientific method (literature analysis, hypothesis generation, and computational simulation).

---

## 2. Core Components

### A. Hypothesis Generation (Co-Scientist Integration)
*   Integrates the multi-agent tournament architecture of Co-Scientist to let AI agents propose, critique, and refine scientific hypotheses.

### B. Computational Discovery (ERA & AlphaEvolve)
*   Powered by **Empirical Research Assistance (ERA)**, which enables AI agents to write, execute, and refactor code in parallel to run thousands of scientific simulation variations.

### C. Literature Insights
*   A NotebookLM-style synthesis interface designed to ingest large corpora of scientific literature, producing comparative analyses, structured data tables, and interactive summaries.

### D. Science Skills
*   A collection of pre-packaged API and data-connector skills integrated into environments like **Google Antigravity**. It allows agents to perform structural bioinformatics, molecular dynamics, and genome analysis by interfacing with databases like UniProt, AlphaFold, and AlphaGenome.
